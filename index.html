• dit clone path-to-repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль git
  • git commit -am "added new file index.html" - зафиксировать изменения и дать им комментарий
  • git push - отправить все зафиксированные изменения в репозиторий на сервер
  • git pull - забрать свежие обновления
  • git add profile.css - добавить новый файл
  • git add . - добавит все файлы
